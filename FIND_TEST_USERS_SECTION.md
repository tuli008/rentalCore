# How to Find "Test Users" Section

## Step-by-Step Navigation

### From OAuth Overview Page:

1. **Look at the left sidebar** - you should see:
   - Overview (you're here)
   - Branding
   - Audience ← **Click this one!**
   - Clients
   - Data access
   - Verification centre
   - Settings

2. **Click "Audience"** in the left sidebar

3. **This opens the "OAuth consent screen" page**

4. **Scroll down on this page** - you'll see sections like:
   - App information
   - App domain
   - Authorized domains
   - Developer contact information
   - **Test users** ← **This is what you need!**

### Alternative Path:

If you don't see "Audience" in the sidebar:

1. Click the hamburger menu (☰) at the top left if sidebar is collapsed
2. Go to: **APIs & Services** → **OAuth consent screen**
3. Scroll down to find **"Test users"** section
